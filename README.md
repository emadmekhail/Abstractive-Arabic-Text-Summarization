# Abstractive-Arabic-Text-Summarization
Arabic Text Summarization using Transformer models (AraBERT, mT5)
**Tools:** Python, PyTorch, Hugging Face Transformers, AraBERT, mT5, REST API

**Description:**  
Developed an abstractive Arabic text summarization system using Transformer-based models.  
- Fine-tuned **AraBERT** and **mT5** for high-quality, human-like summaries.  
- Applied advanced Arabic text preprocessing: cleaning, normalization, tokenization.  
- Evaluated model performance using **ROUGE metrics**.  
- Enabled users to input Arabic text and receive concise summaries through an API.
